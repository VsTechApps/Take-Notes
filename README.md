## Getting Started

### Prerequisites

* [Android Studio](https://developer.android.com/studio)

### Installation

* Fork the repo
* Clone the repo

   ```sh
   git clone https://github.com/your_username/Take-Notes.git
   ```

* Open project from Android Studio and you are good to go
